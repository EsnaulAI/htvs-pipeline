# Dependencias mínimas

Estas son las versiones mínimas recomendadas para ejecutar el pipeline.

## Entorno base (Python + Snakemake)

- python >= 3.10
- snakemake >= 7.32.4
- pandas >= 1.5.0
- biopython >= 1.79
- rdkit >= 2023.03.1
- jupyterlab >= 4.0.0
- openbabel >= 3.1.1

## Herramientas por módulo

- MAFFT >= 7.505
- fpocket >= 4.0
- Open Babel >= 3.1.1
- MMseqs2 >= 14.7e284
- PyDCA >= 1.0.0
